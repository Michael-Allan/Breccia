Breccia: lightweight markup for outlining and drafting

Breccia (pronounced /brɛ́tʃə/ or /brɛ́tʃɪə/) is a lightweight markup language for outlining and drafting based on hierarchical indentation of points and subpoints.

  • Filename extension `.brec` (/dɒt brɛtʃ/).
  • Media type `text/plain`.
