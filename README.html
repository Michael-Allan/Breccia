Breccia: lightweight markup for outlining and drafting

Breccia (pronounced /brɛ́kɪə/) is a lightweight markup language for point-form outlining and drafting based on lineo-hierarchic elements of text known as a fracta.

  • Filename extension `.brec`.
  • Media type `text/plain`.
  • Character encoding UTF-8.
