Breccia: lightweight markup for outlining and drafting

Breccia is a lightweight markup language for point-form outlining and drafting based on nestable, sentence-like elements of text known as fracta, or fractums. Breccia aims not at rich text, but at logically structured text. For an example of what it looks like, see the language definition.

  • Filename extension `.brec`.
  • Media type `text/plain`.
  • Character encoding UTF-8.
