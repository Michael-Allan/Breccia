Breccia: lightweight markup for outlining and drafting

Breccia (pronounced /brɛ́kɪə/) is a lightweight markup language for outlining and drafting based on hierarchical indentation of points and subpoints.

  • Filename extension `.brec`.
  • Media type `text/plain`.
